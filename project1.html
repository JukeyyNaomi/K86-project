Gebruikte software

Voorwoord

Op dit moment is Figma de meest gebruikte tool voor prototyping en UI design. In het hergebruiken van componenten en het gebruik maken van design systems loopt Figma op dit moment op kop. Om deze reden wilde ik graag zelf een design system leren maken om beter te begrijpen hoe dit in elkaar zit en hoe je hier het beste mee kan werken. Dit is een relevante skill voor later in het werkveld.

Onderzoek

Bij het onderzoeken heb ik vooral veel gelezen en naar best practices gekeken. Ik kwam snel bij het boek Atomic design van Brad Frost die uitlegt hoe de opbouw werkt. Dit was een goed startpunt om de kracht van een design system te leren begrijpen. De voorbeelden in zijn boek gaven mij veel inzicht over hoe ik zelf een design system op kon gaan bouwen. Dit ben ik gaan doen (fictief) voor Nespresso.

Data system bouwen

Met genoeg kennis op zak ben ik zelf begonnen met een kleinschalig design system maken. Ik heb een voorbeeld genomen aan de opbouw van Brad Frost. Met zogenaamde atoms beginnen en die uitbouwen naar pages. Dit was in het begin een hoop werk, zeker omdat Figma ook nieuw terrein was. Achteraf heeft alles geresulteerd in een klein design system en een nieuwe kijk op design, wat voor mij erg waardevol is.

Design system gebruiken

Toen mijn kleine design system af was werd het zo ontzettend makkelijk om nieuwe pagina’s te designen. Alle gemaakte elementen zijn herbruikbaar waardoor je heel makkelijk alles in een design om kan bouwen. De kracht van design systems werd mij door dit te doen nog duidelijker. Daarnaast was dit ook leuk om te doen. Om jou ook een beter beeld te geven van de kracht van zo’n systeem is hier een leuk plaatje om mee af te sluiten.

Tips

  • Voor het leren begrijpen van design systems is mijn beste tip om het boek Atomic Design van Brad Frost te lezen, dit helpt je een heel stuk op weg.
  • Probeer naast de vaardigheid om design systems te leren/kunnen bouwen en gebruiken ook eens anders te kijken naar design. Dit kan best een eye-opener zijn voor hoe vele designs in elkaar zitten.
  • Test je eigen design system bij een medestudent.

Ervaring

Ik voelde me na dit blok erg voldaan. Mede omdat ik het gevoel had dat ik er een hele nuttige skill bij heb geleerd, maar ook dat ik Figma toe heb kunnen voegen aan de lijst van software waar ik mee kan werken. Het was ontzettend leuk om te doen en ik zou het bij iedereen aan kunnen raden. In mijn ogen erg nuttig.
